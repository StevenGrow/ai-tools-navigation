# 📦 批量操作功能使用指南

## 功能概述

批量操作功能允许管理员高效地管理多个工具，包括批量删除和批量修改分类。

## 使用步骤

### 1. 打开管理员面板

1. 以管理员身份登录网站
2. 点击右上角的 "👑 管理面板" 按钮
3. 切换到 "🛠️ 工具管理" 标签页

### 2. 选择工具

**方式一：单个选择**
- 点击每个工具左侧的复选框
- 选中的工具会高亮显示（金色背景）

**方式二：全选**
- 点击 "✓ 全选" 按钮选择当前页面所有工具
- 点击 "✗ 取消" 按钮取消所有选择

### 3. 批量操作

当选择了一个或多个工具后，批量操作工具栏会自动显示：

#### 📁 批量修改分类

1. 选择要修改的工具
2. 点击 "📁 修改分类" 按钮
3. 在弹出的对话框中选择新分类
4. 点击 "确认修改" 完成操作

**支持的分类：**
- 💬 对话助手
- 🎨 绘画
- 🎬 视频
- ✍️ 写作
- 💻 编程
- 🎵 音频

#### 🗑️ 批量删除

1. 选择要删除的工具
2. 点击 "🗑️ 批量删除" 按钮
3. 确认删除操作（会显示前3个工具名称）
4. 点击确认后立即删除

**注意：** 删除操作无法撤销，请谨慎操作！

## 功能特点

### ✨ 智能提示
- 显示已选择的工具数量
- 删除前显示工具名称预览
- 操作成功后显示通知消息

### 🎨 视觉反馈
- 选中的工具有金色高亮背景
- 批量操作工具栏动画显示
- 按钮悬停效果

### 📱 响应式设计
- 移动端自动调整布局
- 按钮大小适配触摸操作
- 工具栏垂直排列（小屏幕）

## 使用场景

### 场景1：清理重复工具
1. 使用搜索功能找到重复的工具
2. 选择需要删除的重复项
3. 批量删除

### 场景2：重新组织分类
1. 筛选某个分类的工具
2. 选择需要移动的工具
3. 批量修改到新分类

### 场景3：清理测试数据
1. 筛选用户工具或系统工具
2. 全选当前筛选结果
3. 批量删除

## 快捷操作

| 操作 | 快捷方式 |
|------|---------|
| 全选 | 点击 "✓ 全选" 按钮 |
| 取消选择 | 点击 "✗ 取消" 按钮 |
| 快速删除 | 选择后直接点击 "🗑️ 批量删除" |

## 安全机制

### 确认提示
- 批量删除前需要确认
- 显示将要删除的工具名称
- 明确提示操作不可撤销

### 权限控制
- 只有管理员可以使用批量操作
- 普通用户看不到批量操作按钮
- 数据库层面的权限验证

## 技术实现

### 前端功能
- 使用 Set 数据结构管理选中状态
- 实时更新 UI 反馈
- 动画效果提升用户体验

### 后端操作
- 使用 Supabase 的批量操作 API
- `.in()` 方法批量查询/更新/删除
- 事务性操作保证数据一致性

## 常见问题

### Q: 为什么看不到批量操作工具栏？
A: 需要先选择至少一个工具，工具栏才会显示。

### Q: 批量删除后可以恢复吗？
A: 不可以，删除操作是永久性的，请谨慎操作。

### Q: 可以跨分类批量操作吗？
A: 可以，批量操作不受分类筛选限制。

### Q: 最多可以选择多少个工具？
A: 理论上没有限制，但建议分批操作大量工具。
